# CareerYatra – Internship Portal System

## 📌 Overview

CareerYatra is a full-stack internship portal designed to connect students with recruiters through a centralized platform. The system allows students to explore internships, apply online, manage profiles, and receive skill-based recommendations, while recruiters and admins can efficiently manage internships and platform activities.

---

# 🚀 Features

## 🔐 Authentication & Authorization

* JWT-based authentication
* Secure login and registration
* Password hashing using bcrypt
* Role-based access control (Student, Recruiter, Admin)

## 👨‍🎓 Student Features

* Student profile management
* Browse internships
* Apply for internships
* Skill-based internship recommendations
* CV Generator with downloadable PDF
* Career advice and articles

## 🏢 Recruiter Features

* Recruiter verification system
* Post and manage internships

## 🛠️ Admin Features

* Admin dashboard with platform statistics
* Recruiter management
* User and internship monitoring
* Centralized system management

## 🔔 Additional Features

* Notification system
* Blog/article management
